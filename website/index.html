WHAT IS IT

A subtitle desk for people who actually need the words.

Transcript mode

Listen to audio and show it as text in the same language. Great for English study, interviews, lessons, and meetings.

Translate mode

Listen in one language and show the result in another. Useful when you want support without leaving the live flow.

Floating overlay

Keep the captions in a separate window so you can follow the text while your video, class, or meeting keeps running.

History and export

Keep a floating history window and save a transcript session when you want to study later or review key phrases.

Machine-aware setup

The app reads the PC and suggests a starting preset so weaker machines stay usable and stronger machines get cleaner text.

GPU when available

If the right NVIDIA runtime is present, the app can use GPU acceleration. If not, it falls back to CPU automatically.

HOW TO USE IT

Six simple steps from silent screen to readable captions.

01

Install or open the app

Use the installer if you want the smoothest experience. Open the packaged app if you only want to test quickly.

02

Pick the source

Choose system audio for videos and calls, microphone for speaking practice, or both if you want to capture each side.

03

Choose Transcript or Translate

Transcript keeps the same language. Translate listens in one language and shows the result in another.

04

Set the languages

Choose the source language you expect to hear. If you are translating, choose the target language you want to read.

05

Pick the speed and quality balance

Fast is lighter, Balanced is the default sweet spot, and Accurate is cleaner but slower on weaker machines.

06

Preview, then start

Check the overlay position first, then start the session and let the app handle the caption flow in the background.

CONTROL GUIDE

What each main option actually does.

No jargon, just the decision you need to make.

Source

Choose where the sound comes from: the PC, the microphone, or both.

Mode

Choose Transcript to keep the spoken language, or Translate to read it in a different language.

Source language

Tell the app what language it should expect. This usually helps quality.

Target language

Only used in Translate mode. This is the language you want to read.

Preset

Fast, Balanced, or Accurate. Think of it as speed versus polish.

Model

The speech model behind the captions. Larger models are usually cleaner, but also heavier.

Voice cleanup

Improves rough or noisy audio before transcription. Light is the safest default. Strong is for harder audio.

History overlay

Shows recent caption blocks in a separate floating window.

Auto-save transcript

Writes each block to a file while the session is running.

FAQ

Quick answers before you install.

Is it free?

Yes. It is distributed as a free Windows app download.

Does it need a GPU?

No. GPU can help, but the app falls back to CPU when GPU runtime is not available.

Does translation mode work offline?

Not in this version. Transcription is local, but translation currently uses an online translation step.

Can I save what was said?

Yes. You can enable auto-save or export the session manually from the app.
