What this app is for

Read what you hear while the audio keeps playing.

Whisper Live Subtitles listens to audio from your computer or microphone and shows it as live text. You can use it to understand videos, online classes, meetings, podcasts, or your own speaking practice.

The app has two main modes:

  • Transcript: hear a language and show the same language as text.
  • Translate: hear one language and show the result in another language.

Quick start

Use the app in five easy steps.

  1. Open the app.
  2. Choose Source: system audio, microphone, or both.
  3. Choose Mode: Transcript or Translate.
  4. Pick the language settings and a preset.
  5. Use Preview overlay, then press Start session.
Recommended first run: start with Source: System audio, Mode: Transcript, Language: English, Preset: Balanced, and Voice cleanup: Light.

Main window

What you are looking at.

Left side

This is where you choose how the app should listen and how it should display the results.

Right side

This area shows runtime status, the machine suggestion, and a quick summary of your current setup.

Floating overlay

This is the subtitle window that stays on top while your video or meeting keeps running.

History overlay

This is a separate floating window for recent blocks. Use it only if you want to read back a few recent lines.

Modes

Transcript or Translate?

Transcript

Use this when you want the same language as the spoken audio.

  • Best for language study
  • Best for listening practice
  • Keeps the original language

Translate

Use this when you want to read a different language than the one you hear.

  • Best for support during meetings or classes
  • You choose a source language and a target language
  • Needs internet for the translation step in this version

Controls

What each option means in plain language.

Source
Choose where the sound comes from. Most people use System audio for videos and meetings.
Mode
Choose whether the app should transcribe or translate.
Source language
The language you expect to hear. If you know it, selecting it usually helps quality.
Target language
Only used in Translate mode. This is the language you want to read.
Preset
Fast is lighter, Balanced is the best default, Accurate is cleaner but slower.
Model
The speech model. Bigger models are often cleaner, but they also need more time and more hardware.
Voice cleanup
Off leaves the audio mostly untouched. Light is the recommended default. Strong helps in noisy audio.
Preview overlay
Shows the floating captions window so you can place it before going live.
History overlay
Shows recent transcript blocks in a separate floating window.
Auto-save transcript
Writes each block to a file during the session.

Preset guide

Which preset should you pick?

Fast

Best when your PC is older or you want lighter processing.

Balanced

The best starting point for most people. Good mix of speed and readability.

Accurate

Use this when your PC is stronger and you want cleaner results, especially for study sessions.

If you are not sure, use Apply recommendation. The app reads your computer and suggests a safer starting point.

History and saving

How to keep what was said.

  • History overlay: shows recent blocks in a floating window.
  • Auto-save transcript: writes blocks to disk while the session is running.
  • Save session: exports the full current session when you want to keep it.
  • Clear history: resets the current transcript list inside the app.

Hardware notes

GPU or CPU?

The app tries to use the best runtime it can find. If the right NVIDIA CUDA runtime is available, the app can use the GPU. If not, it falls back to CPU.

  • CPU only: the app still works.
  • GPU available: the app can process heavier models more comfortably.
  • No need to guess: the runtime area in the app tells you what is active.

Troubleshooting

When something does not look right.

No text appears
Check the source, verify that audio is actually playing, and try Preview overlay to confirm the subtitle window is visible.
Text is too slow
Try Balanced or Fast, or reduce cleanup from Strong to Light.
Words are not exact
Choose the correct source language, use a stronger preset if your PC can handle it, and avoid noisy audio when possible.
Translation does not appear
Make sure Mode is set to Translate and that you have selected a target language. This mode also needs internet access.

Final note

Start simple, then tune later.

You do not need to understand every setting on day one. Start with a safe preset, confirm the source, and let the app do the rest. Once it feels good, you can try stronger models, history saving, or translation mode.
